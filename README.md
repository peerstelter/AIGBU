# AIGBU

**KI-gestützte Gefährdungsbeurteilungen aus Bild & Video**

AIGBU automatisiert die Erstellung von Gefährdungsbeurteilungen (GBU) aus Foto- und Videoaufnahmen von Arbeitsstätten. KI analysiert die Bilder, erkennt Gefahrenpotenziale und erstellt daraus strukturierte Dokumente im GEDOKU-Format.

---

## Features

- **Vision-KI Analyse** — automatische Erkennung von Gefahrenquellen, fehlender PSA, unsicheren Zuständen aus Fotos und Videos
- **GEDOKU-Export** — direkte Ausgabe im standardisierten Gefährdungsdokumentations-Format
- **Bild & Video Input** — Einzelfotos, Bildserien und Videoclips werden verarbeitet
- **LLM-gestützte Textgenerierung** — Maßnahmenbeschreibungen und Bewertungen werden automatisch formuliert
- **Bearbeitungs-UI** — generierte GBU kann vor dem Export geprüft und angepasst werden
- **Batch-Verarbeitung** — mehrere Arbeitsstätten in einem Durchlauf auswertbar
